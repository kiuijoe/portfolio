PatchPal

CVE tracker for Mac

A native macOS app for tracking CVEs from the National Vulnerability Database (NVD).

For devs, sysadmins, and security analysts needing a clear view of vulnerabilities affecting their software stack.

  • Save products using their CPE identifiers
  • Monitor newly published CVEs
  • Quickly triage vulnerabilities

PatchPal is built for manual tracking and analysis, avoiding system scanning and privileged access.

Lightweight and built natively with SwiftUI. Designed to be efficient and get out of the way.

Download PatchPal 0.3.1 Beta for macOS (.zip, ~3 MB)
Requires macOS 26 Tahoe. Unzip and move the app to Applications.
No installer. No system access required. No data collection.
