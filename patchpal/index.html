PatchPal

CVE tracker for Mac

Track CVEs from the NVD with a native macOS app.

Find low-hanging fruit.
Group CVEs by the version they are fixed in, to see when a minor update will do, or if a major upgrade is needed.

Triage easier.
Save products using their CPE identifier. Suppress irrelevant CVEs and flag ones requiring action.

Readable, actionable CVEs.
CVE details in a digestible format, distilled down to the details that matter to you.

Minimal by design.
No system scanning or permissions. Built for manual tracking and analysis, when heavy tooling feels unnecessary.

Lightweight and fast.
Built natively for macOS with no unnecessary bloat. PatchPal respects your privacy, only communicating with the NVD.
